  8  /-!
  9  # Derive handler for `inhabited` instances
 10  
 11  This file introduces a derive handle to automatically generate `inhabited`
 12  instances for structures and inductives. We also add various `inhabited`
 13  instances for types in the core library.
 14  -/
 15  
 16  namespace tactic
 17  
 18  /--
 19  Tries to derive an `inhabited` instance for inductives and structures.
 20  
 21  For example:
 22  ```
 23  @[derive inhabited]
 24  structure foo :=
 25  (a : ℕ := 42)
 26  (b : list ℕ)
 27  ```
 28  Here, `@[derive inhabited]` adds the instance `foo.inhabited`, which is defined as
 29  `⟨⟨42, default (list ℕ)⟩⟩`.  For inductives, the default value is the first constructor.
 30  
 31  If the structure/inductive has a type parameter `α`, then the generated instance will have an
 32  argument `inhabited α`, even if it is not used.  (This is due to the implementation using
 33  `instance_derive_handler`.)
 34  -/
