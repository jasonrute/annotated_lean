 44  /-- Apply a function to some local assumptions which are either equalities or
 45      inequalities. For instance, if the context contains `h : a = b` and
 46      some function `f` then `apply_fun f at h` turns `h` into `h : f a = f b`.
 47      When the assumption is an inequality `h : a ≤ b`, a side goal `monotone f`
 48      is created, unless this condition is provided using
 49      `apply_fun f at h using P` where `P : monotone f`, or the `mono` tactic can
 50      prove it. -/
