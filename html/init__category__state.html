126  /-- Adapt a monad stack, changing the type of its top-most state.
127  
128      This class is comparable to [Control.Lens.Zoom](https://hackage.haskell.org/package/lens-4.15.4/docs/Control-Lens-Zoom.html#t:Zoom), but does not use lenses (yet?), and is derived automatically for any transformer implementing `monad_functor`.
129  
130      For zooming into a part of the state, the `split` function should split σ into the part σ' and the "context" σ'' so that the potentially modified σ' and the context can be rejoined by `join` in the end.
131      In the simplest case, the context can be chosen as the full outer state (ie. `σ'' = σ`), which makes `split` and `join` simpler to define. However, note that the state will not be used linearly in this case.
132  
133      Example:
134      ```
135      def zoom_fst {α σ σ' : Type} : state σ α → state (σ × σ') α :=
136      adapt_state id prod.mk
137      ```
138  
139      The function can also zoom out into a "larger" state, where the new parts are supplied by `split` and discarded by `join` in the end. The state is therefore not used linearly anymore but merely affinely, which is not a practically relevant distinction in Lean.
140  
141      Example:
142      ```
143      def with_snd {α σ σ' : Type} (snd : σ') : state (σ × σ') α → state σ α :=
144      adapt_state (λ st, ((st, snd), ())) (λ ⟨st,snd⟩ _, st)
145      ```
146  
147      Note: This class can be seen as a simplification of the more "principled" definition
148      ```
149      class monad_state_functor (σ σ' : out_param (Type u)) (n n' : Type u → Type u) :=
150      (map {} {α : Type u} : (∀ {m : Type u → Type u} [monad m], state_t σ m α → state_t σ' m α) → n α → n' α)
151      ```
152      which better describes the intent of "we can map a `state_t` anywhere in the monad stack".
153      If we look at the unfolded type of the first argument `∀ m [monad m], (σ → m (α × σ)) → σ' → m (α × σ')`, we see that it has the lens type `∀ f [functor f], (α → f α) → β → f β` with `f` specialized to `λ σ, m (α × σ)` (exercise: show that this is a lawful functor). We can build all lenses we are insterested in from the functions `split` and `join` as
154      ```
155      λ f _ st, let (st, ctx) := split st in
156                (λ st', join st' ctx) <$> f st
157      ```
158      -/
