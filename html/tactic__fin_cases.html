 85  /--
 86  `fin_cases h` performs case analysis on a hypothesis of the form
 87  `h : A`, where `[fintype A]` is available, or
 88  `h ∈ A`, where `A : finset X`, `A : multiset X` or `A : list X`.
 89  
 90  `fin_cases *` performs case analysis on all suitable hypotheses.
 91  
 92  As an example, in
 93  ```
 94  example (f : ℕ → Prop) (p : fin 3) (h0 : f 0) (h1 : f 1) (h2 : f 2) : f p.val :=
 95  begin
 96    fin_cases *; simp,
 97    all_goals { assumption }
 98  end
 99  ```
100  after `fin_cases p; simp`, there are three goals, `f 0`, `f 1`, and `f 2`.
101  -/
