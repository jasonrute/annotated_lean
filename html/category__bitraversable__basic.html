 13  # Bitraversable type class
 14  
 15  Type class for traversing bifunctors. The concepts and laws are taken from
 16  <https://hackage.haskell.org/package/base-4.12.0.0/docs/Data-Bitraversable.html>
 17  
 18  Simple examples of `bitraversable` are `prod` and `sum`. A more elaborate example is
 19  to define an a-list as:
 20  
 21  ```
 22  def alist (key val : Type) := list (key × val)
 23  ```
 24  
 25  Then we can use `f : key → io key'` and `g : val → io val'` to manipulate the `alist`'s key
 26  and value respectively with `bitraverse f g : alist key val → io (alist key' val')`
 27  
 28  ## Main definitions
 29    * bitraversable - exposes the `bitraverse` function
 30    * is_lawful_bitraversable - laws similar to is_lawful_traversable
 31
