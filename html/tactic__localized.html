 10  # Localized notation
 11  
 12  This consists of two user-commands which allow you to declare notation and commands localized to a namespace.
 13  
 14  * Declare notation which is localized to a namespace using:
 15  ```
 16  localized "infix ` ⊹ `:60 := my_add" in my.add
 17  ```
 18  * After this command it will be available in the same section/namespace/file, just as if you wrote `local infix ` ⊹ `:60 := my_add`
 19  * You can open it in other places. The following command will declare the notation again as local notation in that section/namespace/files:
 20  ```
 21  open_locale my.add
 22  ```
 23  * More generally, the following will declare all localized notation in the specified namespaces.
 24  ```
 25  open_locale namespace1 namespace2 ...
 26  ```
 27  * You can also declare other localized commands, like local attributes
 28  ```
 29  localized "attribute [simp] le_refl" in le
 30  ```
 31  The code is inspired by code from Gabriel Ebner from the hott3 repository.
 32  -/
