 14  # Single-object category
 15  
 16  Single object category with a given monoid of endomorphisms.  It is defined to facilitate transfering
 17  some definitions and lemmas (e.g., conjugacy etc.) from category theory to monoids and groups.
 18  
 19  ## Main definitions
 20  
 21  Given a type `Î±` with a monoid structure, `single_obj Î±` is `unit` type with `category` structure
 22  such that `End (single_obj Î±).star` is the monoid `Î±`.  This can be extended to a functor `Mon â¥¤
 23  Cat`.
 24  
 25  If `Î±` is a group, then `single_obj Î±` is a groupoid.
 26  
 27  An element `x : Î±` can be reinterpreted as an element of `End (single_obj.star Î±)` using
 28  `single_obj.to_End`.
 29  
 30  ## Implementation notes
 31  
 32  - `category_struct.comp` on `End (single_obj.star Î±)` is `flip (*)`, not `(*)`. This way
 33    multiplication on `End` agrees with the multiplication on `Î±`.
 34  
 35  - By default, Lean puts instances into `category_theory` namespace instead of
 36    `category_theory.single_obj`, so we give all names explicitly.
