 71  /-- Run a monad stack to completion.
 72      `run` should be the composition of the transformers' individual `run` functions.
 73      This class mostly saves some typing when using highly nested monad stacks:
 74      ```
 75      @[reducible] def my_monad := reader_t my_cfg $ state_t my_state $ except_t my_err id
 76      -- def my_monad.run {α : Type} (x : my_monad α) (cfg : my_cfg) (st : my_state) := ((x.run cfg).run st).run
 77      def my_monad.run {α : Type} (x : my_monad α) := monad_run.run x
 78      ```
 79      -/
