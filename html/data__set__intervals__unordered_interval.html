 10  # Intervals without endpoints ordering
 11  
 12  In any decidable linear order `α`, we define the set of elements lying between two elements `a` and
 13  `b` as `Icc (min a b) (max a b)`.
 14  
 15  `Icc a b` requires the assumption `a ≤ b` to be meaningful, which is sometimes inconvenient. The
 16  interval as defined in this file is always the set of things lying between `a` and `b`, regardless
 17  of the relative order of `a` and `b`.
 18  
 19  For real numbers, `Icc (min a b) (max a b)` is the same as `segment a b`.
 20  
 21  ## Notation
 22  
 23  We use the localized notation `[a, b]` for `interval a b`. One can open the locale `interval` to
 24  make the notation available.
 25
