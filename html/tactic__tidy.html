 82  /-- Use a variety of conservative tactics to solve goals.
 83  `tidy?` reports back the tactic script it found.
 84  The default list of tactics is stored in `tactic.tidy.default_tidy_tactics`.
 85  This list can be overridden using `tidy { tactics := ... }`.
 86  (The list must be a `list` of `tactic string`, so that `tidy?`
 87  can report a usable tactic script.) 
 88  
 89  Tactics can also be added to the list by tagging them (locally) with the
 90  `[tidy]` attribute. -/
