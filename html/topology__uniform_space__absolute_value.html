 11  # Uniform structure induced by an absolute value
 12  
 13  We build a uniform space structure on a commutative ring `R` equipped with an absolute value into
 14  a linear ordered field `𝕜`. Of course in the case `R` is `ℚ`, `ℝ` or `ℂ` and
 15  `𝕜 = ℝ`, we get the same thing as the metric space construction, and the general construction
 16  follows exactly the same path.
 17  
 18  ## Implementation details
 19  
 20  Note that we import `data.real.cau_seq` because this is where absolute values are defined, but
 21  the current file does not depend on real numbers. TODO: extract absolute values from that
 22  `data.real` folder.
