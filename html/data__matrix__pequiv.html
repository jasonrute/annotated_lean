  8  # partial equivalences for matrices
  9  
 10  Using partial equivalences to represent matrices.
 11  This file introduces the function `pequiv.to_matrix`, which returns a matrix containing ones and
 12  zeros. For any partial equivalence `f`, `f.to_matrix i j = 1 ↔ f i = some j`.
 13  
 14  The following important properties of this function are proved
 15  `to_matrix_trans : (f.trans g).to_matrix = f.to_matrix ⬝ g.to_matrix`
 16  `to_matrix_symm  : f.symm.to_matrix = f.to_matrixᵀ`
 17  `to_matrix_refl : (pequiv.refl n).to_matrix = 1`
 18  `to_matrix_bot : ⊥.to_matrix = 0`
 19  
 20  This theory gives the matrix representation of projection linear maps, and their right inverses.
 21  For example, the matrix `(single (0 : fin 1) (i : fin n)).to_matrix` corresponds to the the ith
 22  projection map from R^n to R.
 23  
 24  Any injective function `fin m → fin n` gives rise to a `pequiv`, whose matrix is the projection
 25  map from R^m → R^n represented by the same function. The transpose of this matrix is the right
 26  inverse of this map, sending anything not in the image to zero.
 27  
 28  ## notations
 29  
 30  This file uses the notation ` ⬝ ` for `matrix.mul` and `ᵀ` for `matrix.transpose`.
