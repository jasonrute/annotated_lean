111  /-- Without loss of generality: reduces to one goal under variables permutations.
112  
113  Given a goal of the form `g xs`, a predicate `p` over a set of variables, as well as variable
114  permutations `xs_i`. Then `wlog` produces goals of the form
115  
116  The case goal, i.e. the permutation `xs_i` covers all possible cases:
117    `⊢ p xs_0 ∨ ⋯ ∨ p xs_n`
118  The main goal, i.e. the goal reduced to `xs_0`:
119    `(h : p xs_0) ⊢ g xs_0`
120  The invariant goals, i.e. `g` is invariant under `xs_i`:
121    `(h : p xs_i) (this : g xs_0) ⊢ gs xs_i`
122  
123  Either the permutation is provided, or a proof of the disjunction is provided to compute the
124  permutation. The disjunction need to be in assoc normal form, e.g. `p₀ ∨ (p₁ ∨ p₂)`. In many cases
125  the invariant goals can be solved by AC rewriting using `cc` etc.
126  
127  Example:
128    On a state `(n m : ℕ) ⊢ p n m` the tactic `wlog h : n ≤ m using [n m, m n]` produces the following
129    states:
130      `(n m : ℕ) ⊢ n ≤ m ∨ m ≤ n`
131      `(n m : ℕ) (h : n ≤ m) ⊢ p n m`
132      `(n m : ℕ) (h : m ≤ n) (this : p n m) ⊢ p m n`
133  
134  `wlog` supports different calling conventions. The name `h` is used to give a name to the introduced
135  case hypothesis. If the name is avoided, the default will be `case`.
136  
137  (1) `wlog : p xs0 using [xs0, …, xsn]`
138    Results in the case goal `p xs0 ∨ ⋯ ∨ ps xsn`, the main goal `(case : p xs0) ⊢ g xs0` and the
139    invariance goals `(case : p xsi) (this : g xs0) ⊢ g xsi`.
140  
141  (2) `wlog : p xs0 := r using xs0`
142    The expression `r` is a proof of the shape `p xs0 ∨ ⋯ ∨ p xsi`, it is also used to compute the
143    variable permutations.
144  
145  (3) `wlog := r using xs0`
146    The expression `r` is a proof of the shape `p xs0 ∨ ⋯ ∨ p xsi`, it is also used to compute the
147    variable permutations. This is not as stable as (2), for example `p` cannot be a disjunction.
148  
149  (4) `wlog : R x y using x y` and `wlog : R x y`
150    Produces the case `R x y ∨ R y x`. If `R` is ≤, then the disjunction discharged using linearity.
151    If `using x y` is avoided then `x` and `y` are the last two variables appearing in the
152    expression `R x y`. -/
