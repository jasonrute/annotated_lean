  6  A data type for semiquotients, which are classically equivalent to
  7  nonempty sets, but are useful for programming; the idea is that
  8  a semiquotient set `S` represents some (particular but unknown)
  9  element of `S`. This can be used to model nondeterministic functions,
 10  which return something in a range of values (represented by the
 11  predicate `S`) but are not completely determined.
 12  -/

 15  /-- A member of `semiquot α` is classically a nonempty `set α`,
 16    and in the VM is represented by an element of `α`; the relation
 17    between these is that the VM element is required to be a member
 18    of the set `s`. The specific element of `s` that the VM computes
 19    is hidden by a quotient construction, allowing for the representation
 20    of nondeterministic functions. -/
