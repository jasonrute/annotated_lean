 10  # W types
 11  
 12  Given `α : Type` and `β : α → Type`, the W type determined by this data, `W β`, is the inductively
 13  defined type of trees where the nodes are labeled by elements of `α` and the children of a node
 14  labeled `a` are indexed by elements of `β a`.
 15  
 16  This file is currently a stub, awaiting a full development of the theory. Currently, the main
 17  result is that if `α` is an encodable fintype and `β a` is encodable for every `a : α`, then `W β`
 18  is encodable. This can be used to show the encodability of other inductive types, such as those
 19  that are commonly used to formalize syntax, e.g. terms and expressions in a given language. The
 20  strategy is illustrated in the example found in the file `prop_encodable` in the `archive/examples`
 21  folder of mathlib.
 22  -/
