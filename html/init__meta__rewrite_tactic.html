 16  /-- Rewrite the expression `e` using `h`.
 17      The unification is performed using the transparency mode in `cfg`.
 18      If `cfg.approx` is `tt`, then fallback to first-order unification, and approximate context during unification.
 19      `cfg.new_goals` specifies which unassigned metavariables become new goals, and their order.
 20      If `cfg.instances` is `tt`, then use type class resolution to instantiate unassigned meta-variables.
 21      The fields `cfg.auto_param` and `cfg.opt_param` are ignored by this tactic (See `tactic.rewrite`).
 22      It a triple `(new_e, prf, metas)` where `prf : e = new_e`, and `metas` is a list of all introduced meta variables,
 23      even the assigned ones.
