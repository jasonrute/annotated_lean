 12  # The category of elements
 13  
 14  This file defines the category of elements, also known as (a special case of) the Grothendieck construction.
 15  
 16  Given a functor `F : C â¥¤ Type`, an object of `F.elements` is a pair `(X : C, x : F.obj X)`.
 17  A morphism `(X, x) âŸ¶ (Y, y)` is a morphism `f : X âŸ¶ Y` in `C`, so `F.map f` takes `x` to `y`.
 18  
 19  ## Implementation notes
 20  This construction is equivalent to a special case of a comma construction, so this is mostly just
 21  a more convenient API. We prove the equivalence in `category_theory.category_of_elements.comma_equivalence`.
