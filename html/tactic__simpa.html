 13  /--
 14  This is a "finishing" tactic modification of `simp`.
 15  
 16  * `simpa [rules, ...] using e` will simplify the goal and the type of
 17    `e` using `rules`, then try to close the goal using `e`.
 18  
 19  * `simpa [rules, ...]` will simplify the goal using `rules`, then try
 20    to close it using `assumption`. -/
