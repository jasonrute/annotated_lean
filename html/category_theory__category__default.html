 10  /-!
 11  # Categories
 12  
 13  Defines a category, as a type class parametrised by the type of objects.
 14  
 15  ## Notations
 16  
 17  Introduces notations
 18  * `X ⟶ Y` for the morphism spaces,
 19  * `f ≫ g` for composition in the 'arrows' convention.
 20  
 21  Users may like to add `f ⊚ g` for composition in the standard convention, using
 22  ```
 23  local notation f ` ⊚ `:80 g:80 := category.comp g f    -- type as \oo
 24  ```
 25  -/
