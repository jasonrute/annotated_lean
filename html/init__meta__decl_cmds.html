 21  /-- Given a set of constant renamings `replacements` and a declaration name `src_decl_name`, create a new
 22     declaration called `new_decl_name` s.t. its type is the type of `src_decl_name` after applying the
 23     given constant replacement.
 24  
 25     Remark: the new type must be definitionally equal to the type of `src_decl_name`.
 26  
 27     Example:
 28     Assume the environment contains
 29          def f : nat -> nat  := ...
 30          def g : nat -> nat  := f
 31          lemma f_lemma : forall a, f a > 0 := ...
 32  
 33     Moreover, assume we have a mapping M containing `f -> `g
 34     Then, the command
 35          run_command copy_decl_updating_type M `f_lemma `g_lemma
 36     creates the declaration
 37          lemma g_lemma : forall a, g a > 0 := ... -/
