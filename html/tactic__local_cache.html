207  /-- Using the namespace `ns` as its key, when called for the first
208      time `run_once ns t` runs `t`, then saves and returns the result.
209      Upon subsequent invocations in the same tactic block, with the scope
210      of the caching being inherited by child tactic blocks) we return the
211      cached result directly.

213      You can configure the cached scope to be entire `def`/`lemma`s changing
214      the optional cache_scope argument to `cache_scope.def_local`.
215      Note: the caches backing each scope are different.

218      block. -/
