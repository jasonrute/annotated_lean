 96  /-- Adapt a monad stack, changing the type of its top-most environment.
 97  
 98      This class is comparable to [Control.Lens.Magnify](https://hackage.haskell.org/package/lens-4.15.4/docs/Control-Lens-Zoom.html#t:Magnify), but does not use lenses (why would it), and is derived automatically for any transformer implementing `monad_functor`.

100      Note: This class can be seen as a simplification of the more "principled" definition

102      class monad_reader_functor (ρ ρ' : out_param (Type u)) (n n' : Type u → Type u) :=
103      (map {} {α : Type u} : (∀ {m : Type u → Type u} [monad m], reader_t ρ m α → reader_t ρ' m α) → n α → n' α)
104      ```
